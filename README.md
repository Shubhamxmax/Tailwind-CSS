# 💼 Portfolio Website – README

## 📌 Overview

This is my **personal portfolio website** built using **React and Tailwind CSS**. It showcases my profile, skills, education, and contact information with a clean and responsive UI.

---

## 🎯 What I Learned (Tailwind CSS)

* Using **utility-first classes** for fast UI development
* Building **responsive layouts** using Flexbox & Grid
* Applying **custom styling** like colors, spacing, and typography
* Creating **hover effects and transitions**
* Using **clamp() for responsive text sizing**
* Structuring clean and reusable UI components
